# Hackathon-X-Smart-City-2025

My friends and I create the EZPark app is a simple and useful concept designed to help the community in Bandar Pekan by allowing users to digitally pay parking fees and saman fines.
We use Figma for make the prototype of EZPark app.

Protype Link: https://www.figma.com/design/6l0T0p80r65VX6nRD9CZZy/EZPark-Prototype?node-id=1-2&t=vUQcXIowmWfgoXYk-1

Core Solution: EZPark Mobile App
The EZPark mobile app aims to provide a simple and friendly experience for managing parking and paying fines, even for non-tech-savvy users.
